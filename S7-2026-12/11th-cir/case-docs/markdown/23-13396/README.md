# 23-13396 Case Notes

Core CAT funding challenge. This is the most important 11th Circuit case for the concept-release comment because it produced the reported opinion, `Am. Sec. Ass'n v. SEC, 147 F.4th 1264 (11th Cir. 2025)`.

## Use It For

- how ASA/Citadel frame CAT as surveillance, cost burden, and structural overreach;
- how SEC/CAT/SRO intervenors defend Rule 613, the CAT NMS Plan, and funding authority;
- the court's treatment of funding, SRO involvement, agency action, and constitutional arguments;
- citations to private SRO/intervenor financial interests in the funding fight.
