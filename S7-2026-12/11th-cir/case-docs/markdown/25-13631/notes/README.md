# 25-13631 Extraction Notes

Use this subfolder for notes pulled from key filings in this case.

Filled notes are in `high-value-filings-analysis.md`.

For each key filing, create a short note using `key-filing-template.md` and fill only the fields supported by the filing.

## What To Extract

- `claim`: the exact legal or policy claim being advanced.
- `who benefits`: ASA, Citadel, SROs, CAT LLC, SEC, retail investors, regulators, market makers, or other parties.
- `money fact`: fee amount, budget amount, allocation percentage, or cost claim.
- `surveillance/privacy fact`: what data is collected, removed, linked, requested, or accessed.
- `governance fact`: who votes, controls, operates, intervenes, or has a pecuniary interest.
- `court holding`: what the court accepted, rejected, or avoided.
- `comment use`: which concept-release question the point supports.
